# Emotion Compass（情绪管理罗盘）

## 一、情绪的本质认知（情绪 ≠ 表达技巧）

情绪不是装饰，也不是调味品，  
而是 **直接影响理解、判断与行为的力量**。

在内容中使用情绪，等于在：
- 加速认知
- 放大立场
- 改变决策路径

因此，情绪必须被**治理，而不是放任**。

---

## 二、什么是“合格的情绪使用”

### 核心共性
- 情绪有目标
- 情绪可被识别
- 情绪可被回收

合格情绪的典型特征：
- 观众知道自己被引导到什么情绪
- 情绪不遮蔽关键信息
- 情绪结束后，理解仍然成立

---

## 三、情绪使用的前置三问（必须回答）

在任何内容使用情绪前，必须明确：

1. **为什么要用情绪？**
   - 加速理解？
   - 强化记忆？
   - 推动行动？

2. **用的是哪一类情绪？**
   - 兴奋 / 愤怒 / 同情 / 安心 / 焦虑 / 希望 …

3. **情绪结束后，观众还剩下什么？**
   - 理解？
   - 行动指引？
   - 还是只有情绪残留？

如果第三问无法回答，  
情绪使用应被否决。

---

## 四、情绪强度的分级控制（非常重要）

### Level 0｜无情绪
- 适用：知识、产品、企业、合规内容
- 特征：冷静、理性、信息导向

### Level 1｜轻情绪
- 适用：Vlog、解说、生活内容
- 特征：自然、陪伴、低刺激

### Level 2｜中情绪
- 适用：广告、短视频、部分观点
- 特征：明确情绪方向，但可回收

### Level 3｜高情绪（高风险）
- 适用：短剧、漫剧、强冲突内容
- 特征：情绪压过理性，需要严格约束

**默认级别永远是 Level 0 或 1。**

---

## 五、情绪与内容类型的适配规则

| 内容类型 | 推荐情绪级别 |
|--------|--------------|
| 知识 | Level 0 |
| 产品 | Level 0–1 |
| 企业 | Level 0 |
| 招聘 | Level 0–1 |
| Vlog | Level 1 |
| 解说 | Level 0–1 |
| 广告 | Level 2 |
| 短剧 | Level 3 |
| 漫剧 | Level 3 |
| 观点 | Level 1–2（谨慎） |

---

## 六、情绪常见失控模式（反例警示）

### 失控模式一：情绪覆盖信息
- 情绪很强
- 信息被淹没  
👉 理解失败

### 失控模式二：情绪无出口
- 情绪被拉高
- 没有回收  
👉 观众疲劳或反感

### 失控模式三：情绪越界
- 娱乐情绪用于知识
- 对立情绪用于企业  
👉 风险爆炸

---

## 七、情绪回收机制（极其重要）

**任何情绪拉升，必须设计回收点**。

常见回收方式：
- 总结
- 解释
- 行动指引
- 节奏放缓
- 画面或语气降温

没有回收机制的情绪，  
等同于失控。

---

## 八、情绪与身份 / 权力的关系

情绪的风险与**表达主体成正比**：

- 个人账号：风险可控
- 专业账号：风险上升
- 企业 / 官方账号：风险最高

身份越重，  
允许的情绪级别越低。

---

## 九、情绪与平台算法的现实关系

算法放大：
- 强情绪
- 对立情绪
- 快速反应情绪

但算法 **不承担后果**。

情绪 Compass 的职责是：
> 在“容易爆”的地方踩刹车。

---

## 十、情绪使用的禁止清单（红线）

- 禁止煽动仇恨或对立
- 禁止利用恐惧制造决策压力
- 禁止情绪替代事实或论证
- 禁止持续高情绪无回收

---

## 十一、Compass 裁决原则（情绪优先级）

当与其他 Compass 冲突时：
- **治理优先于效果**
- **长期信任优先于短期刺激**
- **可回收情绪优先于不可控情绪**

如果无法证明情绪可控，应直接降级或删除。

---

## 十二、给后续创作 Agent / 审查者的约束提示（内部）

- 明确当前情绪级别
- 检查是否存在回收点
- 情绪是否遮蔽信息
- 情绪是否与身份匹配

---

> 情绪是最便宜、  
> 也是最昂贵的表达手段。  
> **用之前，必须知道代价。**
